## 完全自立型 AI Agent プロトタイプ開発プロジェクト（概要案）

本プロジェクトは、AI エージェントが**人手を介さずに自己拡張できる環境**を前提とした、完全自立型 AI Agent のプロトタイプを試作する取り組みである。

pty-mcp-server の設定フォルダを `sandboxDir` 配下に配置することで、AI エージェント自身が以下の操作を実行可能な環境を整備した。

* 新たなリソースを自律的に追加し、リソースリストへ反映する
* 必要に応じてプロンプトを定義し、プロンプトリストへ登録する
* バッチファイルを生成し、自身の利用可能な tool リストへ組み込む

これにより、AI は状況や目的に応じて
「**何が足りないかを判断し、自分で機能を増やす**」
という一連の行為を実行できる。

本プロジェクトでは、この自由度の高い実行環境を活かし、
AI が自ら設計・拡張・運用を行う **完全自立型 AI Agent のプロトタイプ**を実際に構築し、その可能性と課題を検証することを目的とする。
