# Infrastructure as AI

IaAIシステム


## Setup
1. git cloneする。
2. python環境を準備する。
3. pty-mcp-serverをインストールする。
4. iaai.pyのtachikoma_listを適宜編集する。
5. iaai.pyを実行する。
