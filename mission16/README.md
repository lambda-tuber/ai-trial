# IaAIな日常 FCシリーズ

このシリーズでは、ClaudeにWindows上で、いろいろ作業をしてもらいます。  
Claudeには、様々なペルソナを付与して、一緒に作業してもらいます。

## Setup
1. git cloneする。
2. python環境を準備する。
3. Claudeにpty-mcp-serverをインストールする。
4. Claudeの拡張設定が参照するpty-mcp-server.yamlにおいて、workDirをフルパスでcloneしたフォルダのpty-mcp-serverに指定する。
